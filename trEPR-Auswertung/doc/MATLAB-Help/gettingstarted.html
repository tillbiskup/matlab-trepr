What is the trEPR Toolbox?

The trEPR Toolbox is a collection of MATLAB® scripts for the analysis of transient EPR data recorded with the program fsc2. Most of the functions are independent of the recording format (data file recorded with fsc2).

Introduction

Typically, transient EPR spectra are measured as 2D data sets in the time and field domain. At each point of the field axis a single 1D spectrum over time is recorded (called time trace or time slice). Several of these time slices at different points on the field axis form the complete 2D data set.

Both dimensions (field domain and time domain) are of interest for the analysis of the data. Therefore, the trEPR Toolbox provides functions to get slices from the 2D data set as well in field direction as in time direction.

Functions

For a direct acces there is the alphabetic index of the functions. Especially for users new to this toolbox the overview of the functions grouped by categories may be useful.

User Interface

Currently, most of the functions provided by the trEPR toolbox are designed to be used at the command line. Some of them do their job in an interactive manner asking the user questions, others are more or less quiet.

One exception is the GUI_fsc2_display function that provides a graphical users interface (GUI) similar to that of the fsc2 program used mainly for recording the transient spectra.
