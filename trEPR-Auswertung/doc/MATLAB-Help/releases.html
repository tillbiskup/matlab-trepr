Release Notes

0.1.2

2007-07-27

New features

  • Interactive program UITimeSliceFitNExp to fit time slices in an multiexponential approach.
  • Program save_figure to save an open figure window to an EPS/FIG-file (depending on the platform).
  • Program platform to retrieve platform specific information.
  • Programs UIAskForExistingFileName and UIAskForExistingFileName for modularized handling of user provided file names.
  • Program spemanAscii2trEPR2Ddata to convert speman ASCII export files (FR) to trEPR-2D-datafiles compatible with the trEPR toolbox.

Changes

  • Interactive program compensate_accumulate_2Dspectrum: saves now 2D data together with parameters, improved console output.
  • Interactive program compensate_accumulate_2Dspectrum: improved menu buttons (replaced "exit" with "continue").
  • Deleted old sample_* and script_* files that are no longer used and were depreciated starting with version 0.1.0.
