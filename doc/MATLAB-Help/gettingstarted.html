What is the LaTeX Toolbox?

The LaTeX Toolbox is a collection of MATLAB® functions to enable MATLAB® to create LaTeX output.

Introduction

MATLAB® can produce somehow useful graphical output, but it fails completely with the task to print tabular data in a convenient way. As a user of LaTeX for many years the the idea was quickly at hand: why not to tell MATLAB® how to produce LaTeX output?

The first results are a set of functions collected in this toolbox providing MATLAB® with the ability to produce well-formed LaTeX output from the command line and of course from within other functions.

Functions

For a direct acces there is the alphabetic index of the functions. Especially for users new to this toolbox the list of the functions grouped by categories may be useful.

User Interface

Currently, all of the functions provided by the LaTeX toolbox are designed to be used at the command line.
