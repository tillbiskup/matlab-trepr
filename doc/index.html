The trEPR toolbox is a MATLAB™ toolbox for preprocessing, display, analysis, and postprocessing of transient EPR data.

The toolbox is designed to be centered around a GUI that gives the user convenient access to all functions. Nevertheless, the actual functions can all be used from the MATLAB™ command line.

Getting started

The simplest way to getting started with the trEPR toolbox is to type "trEPRgui" at the MATLAB™ command line:

trEPRgui;

This will open the trEPR GUI for you. No worries, in case that you already opened a trEPR GUI window that got forgotten in the background, the above command will not start another trEPR GUI, but simply bring the existing one back on top of your desktop.

Documentation

The toolbox is extensively documented within the GUI itself. You can access this help either by pressing the button with the question mark, or by hitting F1.

For the most up to date information, please have a look at the trEPR Toolbox Webpage. This is as well the place to go to for updates.
