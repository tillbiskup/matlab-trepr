Information for datasets displayed

There is a wealth of information known to the toolbox for each dataset displayed. To help the user keeping track of the datasets displayed in a figure, this information is collected and displayed in the "Information" panel as well as appended to the caption file.

What get's displayed

The information block starts with an overview of the number of datasets, followed by a block for each dataset.

As there is no need to edit this information, the respective panel is set to read-only. This is indicated consistently throughout the toolbox via the grey rather than white background.

Currently, the information displayed is the same as if one would create an info file for the respective dataset. In the future, both, history records and display options will be added as well.
