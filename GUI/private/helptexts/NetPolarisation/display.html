Display

The display panel provides you with additional controls of the display, as well as (most probably) the possibility to export/save the plot.

Grid

Grids in x and y direction can be added independently, and the minor grid can be added (simultaneously for both axes).

In this particular case, x and y refers to the abscissa and ordinate of the current display, not to the x, y, and z dimensions of the dataset, as everywhere else.

"Zero" toggles between displaying a zero line.

Pretrigger statistics

To help analyse the net polarisation, you can display both mean and standard deviation (as area, mean +/- stdev) of the pretrigger part of the signal independently or together. Additionally, you can display min and max of the pretrigger part of the signal as area.

Export figure

To export the current display to a file, choose the file type (currently Matlab™ FIG, EPS, PDF, PNG) and hit "Export".

The "Format" popupmenu allows to specify a format (aspect ratio, size of the graphics) that makes it easier to use the saved figures, e.g., in reports. These formats can be fully customised using the "fig2file.ini" file in the same directory as the "fig2file" function.
