The trEPR toolbox is a MATLAB™ toolbox for preprocessing, display, analysis, and postprocessing of transient EPR data.

The toolbox is designed to be centered around a GUI that gives the user convenient access to all functions. Nevertheless, the actual functions can all be used from the MATLAB™ command line.
