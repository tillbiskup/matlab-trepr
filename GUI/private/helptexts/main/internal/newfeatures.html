What's new in the trEPR toolbox?

An increasingly detailed list of new features for each version.

Basically, the complete toolbox got reimplemented for version 0.3 (back in 2011), and the GUI rewritten with a complete new layout (and underlying code). Currently, besides adding new functionality, major reorganisation of the underlying code is taking place, aiming at a much more robust (and in certain aspects faster) code basis.
