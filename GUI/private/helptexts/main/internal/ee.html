Procrastination: Display comics

There is a series of comics that can be displayed from within the GUI using the command line feature, "CMD". (If you have no other reason than this, now it is time to get a bit familiar with the CMD).

The following commands may get you started:

show dilbert
The famous comic strip by Scott Adams.
Highly copyrighted, therefore only for private use.
Execute command
 
show phdcomic
For all PhD students...
Execute command
 
show xkcd
A webcomic of romance, sarcasm, math, and language.
Includes the (quite important) tooltips of the original webpage version.
Execute command
 
show nichtlustig
For the German-speaking users, a quite famous German cartoon.
Highly copyrighted, therefore only for private use.
Execute command

Important note: All comics are directly retrieved from their respective websites. Sometimes the websites change, thus breaking the code retrieving the images. Sometimes, the images are in a format that causes trouble to Matlab(r).
