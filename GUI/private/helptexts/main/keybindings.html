Key bindings

There are a couple of key bindings that may make life easier when working with this GUI.

Wherever possible, these same shortcuts apply consistently throughout the whole GUI and all its subwindows.

Switching between panels

Ctrl-1 ... Ctrl-9

(Mac: Ctrl or Command)

Switching GUI modes

c
Scale spectra (in both dimensions)
d
Displace spectra (in both dimensions)
m
Measure (between two points)
p
Pick (single point)
s
Scroll through spectra (in both dimensions)
z
Zoom

Switching display modes

Ctrl-x ... Ctrl-z

(Mac: Ctrl or Command)

Function keys (F1...F12)

F1
Help window
F2
"Help:About" window
F3
Informations about currently active dataset
F4
Accumulate (ACC)
F5
Average (AVG)
F6
Baseline correction (BLC)
F10
"Status" window

Closing the GUI

Ctrl-w

(Mac: Ctrl or Command)

Additional key bindings

Ctrl-i
Informations about currently active dataset
Ctrl-l
Enter command line (located at the bottom of the main GUI)

(Mac: Ctrl or Command)
