Introduction

A rather short introduction to the Command Line of the trEPR GUI.

If in a hurry

If you know the command you want to use, just type it into the command line and press Return. If you're not using the toolbox regularly, just continue using your mouse. This is most probably the much more convenient way for you. If unhappy, RTFM...

To the gentle reader

The Command Line of the trEPR GUI is a rather powerful tool, allowing the user (once it will be implemented properly) to perform most of the tasks by typing commands on the keyboard. This comes on cost of having to remember the commands.

General syntax

In general, you type a <command> that might be followed by one or more additional parameters. Commands are not case sensitive, command and parameter(s) are separated by single spaces.

Command line history

Within one session of the trEPR GUI, the command line history is saved and can be accessed as usual by using the arrow keys.

Getting help for a particular command

If you are interested in help for a specific command of the command line of the trEPR GUI, you can type either help or simply ?, followed by the command <command> you are interested in as a parameter. For example: 

help disp

will display the help for the disp command.

To get an overview of all the commands currently implemented in the command line of the trEPR GUI toolbox, just type 

help commands

and have a look at the list on the left side of the window.
