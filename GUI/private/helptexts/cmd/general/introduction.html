Introduction

A rather short introduction to the Command Line of the trEPR GUI.

If in a hurry

If you know the command you want to use, just type it into the command line and press Return. If you're not using the toolbox regularly, just continue using your mouse. This is most probably the much more convenient way for you. If unhappy, RTFM...

To the gentle reader

The Command Line of the trEPR GUI is a rather powerful tool, allowing the user (once it will be implemented properly) to perform most of the tasks by typing commands on the keyboard. This comes on cost of having to remember the commands.
