set

Set values of GUI appdata fields.

Description

The set command aims mainly at programmers and developers and is used to set values of GUI appdata fields. Therefore, using this command normally requires quite some familiarity with the GUI appdata structure. Handle with care, as you may easily break an existing GUI session (and possibly even crash your Matlab session).

Syntax

set <field> <value>

Options

field
(Cascaded) field of the GUI appdata.
value
Value the field should be set to.
