filter

Apply filter functions in one dimension.

Description

The filter command is used to apply different 1D filter functions to a dataset.

Syntax

filter

filter [<dataset>] <x|y> <type> [width=]<width> [order=<order>] [deriv=<deriv>]

Options

Currently, the filter command takes the following options:

dataset (OPTIONAL)
Number of the dataset to apply the filter to.
If no number is given (i.e., the first argument is not a number, but a letter), averaging is performed using the currently active dataset.
x|y
Dimension along which to perform the filtering.
type
Type of filter function.
Currently, the following filter functions (options) are understood: boxcar, binomial|binom, savitzky-golay|savgol.
width
Width of the filter window.
All values are in points.
order (OPTIONAL)
Order used for the Savitzky-Golay filter.
deriv (OPTIONAL)
Derivative used for the Savitzky-Golay filter.
