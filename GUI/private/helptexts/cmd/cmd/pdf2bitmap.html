pdf2bitmap

Convert PDF file (figure) to bitmap (currently PNG).

Description

The pdf2bitmap command is used to convert PDF files (exported figures) into bitmap files (currently PNG).

Syntax

pdf2bitmap <filename>

The command needs to be called with at least a filename of a PDF file as option.

Options

filename
Name of the (PDF) file that should be converted into a bitmap (PNG) file.
If no extension is provided, ".pdf" is silently added.

Please note: The command relies on the function pdf2bitmap of the trEPR toolbox that itself makes heavy use of the system command and relies on a working ImageMagick installation. Therefore, currently only Linux and Mac platforms are supported.
