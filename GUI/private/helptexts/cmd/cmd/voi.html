voi

Setting "view of interest" (VOI) in characteristics of dataset.

Description

The voi command works in all display modes, but to see what you actually save as VOI, open the 3D representation window using the respective page of the display panel. Without options it adds the currently selected parameters for the 3D representation of the currently active dataset as "view of interest" (VOI) to the list of characteristics of this dataset.

If used with additional options, the voi command sets the particular field of the VOI structure.

Syntax

voi

voi [<label>]

voi <label|comment> <option>

Options

Currently, the voi command takes the following options:

label
Set the label of the current VOI.
comment
Set the comment of the current VOI.

If none of the above listed options is provided, additional text will be handled as label for the VOI.

The command automatically checks whether a VOI for this dataset with the identical parameters has already been created, and if so, it will update the parameters as specified by the options. Therefore, for each set of parameters only one VOI will exist.
