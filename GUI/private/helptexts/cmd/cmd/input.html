input

input dataset.

Description

The input command is used to prompt the user interactively for values, mainly in context of scripting.

Syntax

input <variablename>

input <variablename> [type=<type>]

input <variablename> [preset=<default value>]

input <variablename> [<prompt>]

The command needs at least a variable name.

Options

variablename
Name of the variable.
The variable may only contain alphanumeric characters and the underscore and needs to start with an alphabetic character (as any Matlab® variable).
type
Any Matlab® variable type the variable should be casted to.
default value
Default value that gets displayed in the edit field of the prompt
prompt
Text shown above the edit field of the prompt (default: variable name).
