poi

Setting "point of interest" (POI) in characteristics of dataset.

Description

The poi command works in all display modes, but only if a point has been picked or a distance measured using the measurement panel. Without options it adds the currently selected (picked) point of the currently active dataset as "point of interest" (POI) to the list of characteristics of this dataset.

If used with additional options, the poi command sets the particular field of the POI structure.

Syntax

poi

poi [<label>]

poi <label|comment|display> <option>

Options

Currently, the poi command takes the following options:

label
Set the label of the current POI.
comment
Set the comment of the current POI.
display
Toggle whether the current POI gets displayed in a report. Allowed values are "true", "false", and numbers that will be converted into boolean values according to Matlab's logical command.

If none of the above listed options is provided, additional text will be handled as label for the POI.

The command automatically checks whether a POI for this dataset with the identical coordinates has already been created, and if so, it will update the parameters as specified by the options. Therefore, for each coordinate only one POI will exist.
