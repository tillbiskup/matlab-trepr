clear

Clear variable.

Description

The clear command is used to clear variables from the CMD environment.

Syntax

clear <variablename>

clear all

The command needs at least one option.

Options

variablename
Name of the variable that should be removed from the list of variables.
all
Special keyword removing all variables from the CMD environment.
Note that although you can in general define a variable named "all", it seems not wise to do so, as you cannot explicitly remove it again.
