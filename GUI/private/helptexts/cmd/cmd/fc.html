fc

Set frequency of dataset(s) to given frequency (frequency correction, fc).

Description

The fc command recalculates the magnetic field axis for a given dataset for the given frequency.

If used with additional options, the fc command sets the frequency for the given set of datasets.

Syntax

fc <frequency>

fc [<datasetNumber>] <frequency>

fc [<all|visible|invisible>] <frequency>

Options

Currently, the fc command takes the following options:

frequency
Microwave frequency (in GHz) that shall be set for dataset(s).
datasetNumber
Number of the dataset the command should operate on.
all|visible|invisible
Keyword specifying groups of datasets the command should operate on..
