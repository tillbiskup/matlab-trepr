doi

Setting "distance of interest" (DOI) in characteristics of dataset.

Description

The doi command works in all display modes, but only if a distance has been measured using the measurement panel. Without options it adds the currently selected distance of the currently active dataset as "distance of interest" (DOI) to the list of characteristics of this dataset.

If used with additional options, the doi command sets the particular field of the DOI structure.

Syntax

doi

doi [<label>]

doi <label|comment|display> <option>

Options

Currently, the doi command takes the following options:

label
Set the label of the current DOI.
comment
Set the comment of the current DOI.
display
Toggle whether the current DOI gets displayed in a report. Allowed values are "true", "false", and numbers that will be converted into boolean values according to Matlab's logical command.

If none of the above listed options is provided, additional text will be handled as label for the DOI.

The command automatically checks whether a DOI for this dataset with the identical coordinates has already been created, and if so, it will update the parameters as specified by the options. Therefore, for each coordinate only one DOI will exist.
