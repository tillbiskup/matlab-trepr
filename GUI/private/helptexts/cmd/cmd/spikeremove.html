spikreremove

Remove spikes (artifacts) in datasets using interpolation.

Description

Spikes (artifacts) often result from transient detuning of the cavity during the measurement process, particularly if the background is subtracted already in the transient recorder.

The spikreremove command takes field positions as option(s) and performs interpolation for these field points.

Syntax

spikreremove positions

Options

All options are interpreted as (list of) field positions in the unit of the given dataset.

You can use all the different notation options for vectors Matlab(r) provides, as str2num is used internally to convert the options into a vector of field positions.

The function checks for the positions to exist in the dataset. If there is any problem, the function will issue a warning and return.

Warning

Please note that this function is very close to manipulating data in a bad way and therefore not recommended for general use or data to be published. It is always better to record a dataset again that contains spikes this function can remove.

Furthermore, it cannot be excluded that additional artifacts are introduced by this function. Therefore, use it entirely on your own risk and responsibility.

The fact that this function has been applied to a dataset is documented in the dataset's history - unless some (evil) person removes this history record by purpose.
