remove

Remove datasets from the main GUI.

Description

The remove command is used to remove datasets. Alternatively you could click on the "Hide" button in the "Dataset" panel of the main GUI.

Syntax

remove

remove visible|invisible|all

remove visible|invisible|all force

Options

Currently, the remove command takes the following options:

visible
Remove all currently visible datasets.
invisible
Remove all currently invisible datasets.
all
Remove all currently loaded datasets.
force
Used in combination with any of the above options, ignore unsaved changes to any of the datasets and silently remove.

If called without options, remove will remove the currently active (selected) dataset.

Further Information

The remove command provides you with a convenient way of removing datasets from the GUI, particularly if you want to remove more than one dataset or the invisible datasets.

Please note: Removing datasets means that they are only removed from the GUI, not from the file system. The only way you may encounter data loss is with using the force option or discarding changes you made to datasets you are about to remove from the GUI.
