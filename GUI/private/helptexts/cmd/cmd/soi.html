soi

Setting "slice of interest" (SOI) in characteristics of dataset.

Description

The soi command works only in 1D display modes. Without options it adds the currently displayed slice of the currently active dataset as "slice of interest" (SOI) to the list of characteristics of this dataset.

If used with additional options, the soi command sets the particular field of the SOI structure.

Syntax

soi

soi [<label>]

soi <label|comment|display|avgWindow> <option>

Options

Currently, the soi command takes the following options:

label
Set the label of the current SOI.
comment
Set the comment of the current SOI.
display
Toggle whether the current SOI gets displayed in a report. Allowed values are "true", "false", and numbers that will be converted into boolean values according to Matlab's logical command.
avgWindow
Width of the average window.

If none of the above listed options is provided, additional text will be handled as label for the SOI.

The command automatically checks whether a SOI for this dataset with the identical coordinates has already been created, and if so, it will update the parameters as specified by the options. Therefore, for each coordinate only one SOI will exist.
