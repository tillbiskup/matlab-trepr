?

Show help for the trEPR GUI command line.

Description

The ? command is used to display help for the trEPR GUI command line.

Syntax

?

? <command>

Options

Currently, the ? command takes the following options:

<command>
Show help for the specific command <command>

If called without options, it will just display the help window you're currently seeing.
