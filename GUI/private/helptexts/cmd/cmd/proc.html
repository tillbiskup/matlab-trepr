proc

Perform processing of current or explicitly selected dataset.

Description

The proc command is used to perform basic processing steps of datasets. Alternatively, you can use the GUI elements in the "processing" panel.

Syntax

proc

proc <option>

Options

Currently, the proc command takes the following options:

poc | pretrigger
Perform pretrigger offset compensation.
bgc | background
Perform background compensation.
blc | baseline
Open BLC GUI window.
acc | accumulate
Open ACC GUI window.

If called without a second option, proc performs the processing on the currently active dataset. Alternatively, call with the number of a dataset as second (optional) argument.
