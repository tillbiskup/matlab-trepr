avg

Performing averaging of a 2D dataset in one dimension, alternatively: opening the AVG GUI window.

Description

The avg command without options opens the AVG GUI window of the trEPR GUI. Alternatives would be to press F6 or click on the "AVG" button in the main GUI.

If used with a number of options, the avg command performs averaging of twodimensional datasets in the specified dimension from the command line without using the AVG GUI.

Syntax

avg

avg [<dataset>] <x|y> <start:stop>|<center;width> [<label>]

Options

Currently, the avg command takes the following options:

dataset (OPTIONAL)
Number of the dataset to perform the averaging for.
If no number is given (i.e., the first argument is not a number, but a letter), averaging is performed using the currently active dataset.
x|y
Dimension along which to perform the averaging.
start:stop | center,width
Window to perform the averaging for.
There are two alternative ways to specify this window, either using a colon and specifying start and stop values, or using a comma specifying center value and width.
All values are in units of the dataset the averaging is performed for.
label (OPTIONAL)
Label of the new dataset resulting from the averaging process.
