load

Loading datasets into trEPR GUI

Description

The load command loads datasets into the trEPR GUI. Alternatives would be to press Ctrl+o or click on the "Load" button in the load panel.

Syntax

load

load <filename>

Please note that the file selection dialogue will be opened in case that no filename is given as an option.

The load function respects all settings of the load panel in the GUI.

Options

filename
Name of the file to load.
