help

Display help for different topics or commands.

Description

The help command is used to access the help facilities built-in into the trEPR toolbox and GUI, respectively. Alternatively, you can use the command ? as a shortcut. See the respective help for further information.

Syntax

help

help <option>

If called without option, the trEPR GUI help window will open.

Options

help
Open the "Help" window of the trEPR GUI.
about
Open the "About" window of the trEPR GUI.
modules
Open the "Modules" window of the trEPR GUI displaying information about the modules currently installed, their maintainers, version and other useful stuff.
<command>
Open the help window for the command line of the trEPR GUI and show help for the command <command>.
