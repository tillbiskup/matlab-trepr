convert

Converting units of axes.

Description

The convert command needs at least one option to work.

If used with one option, the convert command converts the given measure for the currently active dataset. If used with two options, the first option is the dataset number.

Syntax

convert

convert [<dataset>] <conversion>

Options

Currently, the convert command takes the following options:

dataset (OPTIONAL)
Number of the dataset to perform the conversion for.
Special keywords are "all", "visible"|"vis", and "invisible"|"invis"|"inv" for all, visible, and invisible datasets, respectively.
If no dataset is given, conversion is done for the currently active dataset.
<conversion>
Conversion of units to be performed. Currently, the following conversions are supported (all conversion options are case insensitive):
G2mT
Convert G into mT
mT2G
Convert mT into G
us2s
Convert us into s
s2us
Convert us into s
