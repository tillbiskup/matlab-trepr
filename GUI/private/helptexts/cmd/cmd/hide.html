hide

Hide the currently active dataset.

Syntax

hide

hide all

Options

Currently, the hide command takes the following options:

all
Hide all visible datasets

If called without options, it will hide the currently active (selected) dataset.

Description

The hide command is used to hide datasets. Alternatively you could click on the "Hide" button in the "Dataset" panel of the main GUI.
