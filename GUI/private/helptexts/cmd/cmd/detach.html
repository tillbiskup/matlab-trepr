detach

Detach the current plot, i.e. replotting it in a separate Matlab window.

Description

The detach command detaches the current plot from the main GUI, i.e. it replots the figure in a separate Matlab window that can be postprocessed as any other Matlab plot window. Alternatively you could click on the "Detach" button in the main GUI.

Syntax

detach

Options

Currently, the detach command takes no options.
