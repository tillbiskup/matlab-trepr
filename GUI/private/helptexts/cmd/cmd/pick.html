pick

Pick minimum or maximum of currently active dataset and set x and y sliders accordingly.

Description

The pick command is used to pick datasets. Alternatively you could click on the "Hide" button in the "Dataset" panel of the main GUI.

Syntax

pick

pick <option>

Options

Currently, the pick command takes the following options:

min
Select the (absolute) minimum of the currently loaded dataset and set scroll sliders in x and y direction accordingly.
max
Select the (absolute) maximum of the currently loaded dataset and set scroll sliders in x and y direction accordingly.

The pick command provides you with a convenient way of easily displaying slices along x and y in the respective minimum and maximum.
