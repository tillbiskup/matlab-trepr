mwfreq

Opening the MW Frequency Drift GUI window.

Description

The mwfreq command opens the MW Frequency Drift GUI window of the trEPR GUI. Alternatively, you could press the respective button in the "Analysis" panel of the trEPR GUI.

Syntax

mwfreq

Options

Currently, the mwfreq command takes no options.
