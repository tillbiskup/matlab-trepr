show

Show the currently active dataset.

Description

The show command is used to show datasets. Alternatively you could click on the "Show" button in the "Dataset" panel of the main GUI.

Syntax

show

show <dataset>

show all

show all visible

show only active

Options

Currently, the show command takes the following options:

dataset
Number of the dataset to make visible and/or active.
all
Make all datasets visible
all visible
Show all visible datasets
only active
Display only the currently active dataset.
To unset this, type show all or show all visible.
next
Display next dataset in the list of visible datasets.
prev
Display previous dataset in the list of visible datasets.

If called without options, it will make visible the dataset currently selected in the list of invisible datasets.
