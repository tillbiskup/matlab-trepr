dir

Show/Set directories used internally in the trEPR GUI.

Description

The dir command is used to display and change the directories used internally in the trEPR GUI for loading, saving and exporting data. Setting each of these directories to a specific value can greatly facilitate the work with the GUI.

Syntax

dir

dir <parameter>

dir <parameter> <value>

If called without parameters, dir opens a window displaying all current directory settings of the trEPR GUI that allows as well to change these.

If called with a parameter only, but without value, the current value of the respective parameter will be displayed in the status window of the trEPR GUI.

Parameters

The option <parameter> following the dir command can currently take the following values:

load|loaddir
Display/set the default directory used for loading datasets. This is the directory that is preselected in the file selection dialogue.
If used followed by a value, the directory is set to this value.
If used without value, the directory in use currently is displayed in the status window.
save|savedir
Display/set the default directory used for saving datasets. This is the directory that is preselected in the file selection dialogue.
If used followed by a value, the directory is set to this value.
If used without value, the directory in use currently is displayed in the status window.
savefig|savefigdir|figsave|figsavedir|fig|figdir
Display/set the default directory used for saving figures. This is the directory that is preselected in the file selection dialogue.
If used followed by a value, the directory is set to this value.
If used without value, the directory in use currently is displayed in the status window.
export|exportdir
Display/set the default directory used for exporting datasets. This is the directory that is preselected in the file selection dialogue.
If used followed by a value, the directory is set to this value.
If used without value, the directory in use currently is displayed in the status window.
