save

Save dataset.

Description

The save command is used to save datasets (in toolbox format).

Syntax

save

save <filename>

save [<id>] <filename>

If the command is called without option, the currently active dataset is saved using a filename retrieved by the information availabe.

Options

filename
Name of the file the dataset should be saved to.
As file extension, automatically ".tez" is appended and a given extension, if any, silently removed.
File names may therefore neither contain dots nor spaces.
id
Operate on the dataset with the given ID.
