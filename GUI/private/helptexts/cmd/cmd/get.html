get

Display values of GUI appdata fields.

Description

The get command aims mainly at programmers and developers and is used to display values of GUI appdata fields. Therefore, using this command normally requires quite some familiarity with the GUI appdata structure. The value of the given field is displayed in the GUI status window.

Syntax

get <field>

Options

field
(Cascaded) field of the GUI appdata.
