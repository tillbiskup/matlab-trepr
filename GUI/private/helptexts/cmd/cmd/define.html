define

Define variables on the CMD line.

Description

The define command needs at least one option to work.

If used with one option, the define command assigns the value after the assignment character ("=") sign in the option to the variable with the name given before the assignment character in the option.

Variables can be used within commands by prepending the variable name with a "$".

Syntax

define <variable>=<value>

define <variable> = <value>

Options

Currently, the define command takes the following options:

<variable>=<value>
Assign the value <value> to the variable <variable>

You can use as many white characters as you wish within the assignment. Both value and argument are trimmed meaning that they won't contain whitespace characters at their start or end afterwards.

Hint: Variable definitions may contain variables as well. (You're solely responsible for retaining or loosing overview of the assignments.) In this case it is necessary to surround the variable with spaces for correct parsing.
