pause

Halt execution for a given delay.

Description

The pause command aims mainly at programmers and developers and is used to halt execution of a script (and, more generally, Matlab itself) for a given delay. (It serves as a wrapper for the Matlab function of the same name.)

Syntax

pause <delay>

Options

delay
Time to halt execution in seconds.
Negative times are not allowed and result in a warning.
